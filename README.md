# Smart Safety System for Two Wheelers

An IoT-based safety solution developed using ESP32, Python, and multiple sensors to enhance two-wheeler rider safety through helmet enforcement, accident detection, obstacle monitoring, GPS tracking, emergency alerts, and smart navigation assistance.

## Features

* Helmet Detection using FSR sensor
* Accident Detection using MPU6050 accelerometer and gyroscope
* GPS-based real-time location tracking
* Automatic WhatsApp emergency alerts
* Obstacle detection using ultrasonic sensor
* Automatic headlight control using LDR
* Navigation assistance using Google Directions API
* LCD-based direction and distance display
* LED-based turn indication system

## Hardware Components

* ESP32 Microcontroller
* MPU6050
* Neo GPS Module
* HC-SR04 Ultrasonic Sensor
* Force Sensitive Resistor (FSR)
* LDR Sensor
* LCD Display (16x2)
* LEDs
* Buzzer
* Push Buttons

## Software Components

* Arduino IDE
* Python
* Google Directions API
* WhatsApp Web Automation
* TinyGPS++
